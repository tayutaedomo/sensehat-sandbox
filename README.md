# sensehat-sandbox
Try Sense HAT with Raspberry Pi

## Setup
```
$ git clone git@github.com:tayutaedomo/sensehat-sandbox.git
$ cd sensehat-sandbox
```


## Fonts
```
$ python3 fonts/font8x8.py
```

```
$ python3 fonts/font4x8.py
```


## LED
```
$ python3 led/color_edit.py
```

```
$ python3 led/rainbow.py
```
